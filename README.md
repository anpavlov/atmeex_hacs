# Atmeex Cloud

[English version](readme.en.md)

Интеграция для Home Assistant, позволяющая управлять бризерами Airnanny A7 через облачный сервис Atmeex.

## Возможности

- Управление режимами работы: выключено, нагрев, вентиляция
- 7 скоростей вентилятора
- 3 позиции заслонки: приток, смешанный режим, рециркуляция
- Управление температурой нагрева (10-30°C, шаг 0.5°C)
- Пассивное проветривание (заслонка открыта, вентилятор выключен)
- Авторизация по email + пароль, либо по номеру телефона с одноразовым кодом из СМС

## Поддерживаемые устройства

- Airnanny A7

## Установка

## Платформы

Интеграция создаёт четыре типа сущностей для каждого устройства:

- **Climate** — полное управление бризером: питание, температура, скорость вентилятора, режим заслонки
- **Switch** — переключатель питания с автоматическим управлением заслонкой
- **Fan** — управление скоростью вентилятора и питанием БЕЗ управления заслонкой
- **Select** — выбор положения заслонки (открыта/смешанная/закрыта)

### Climate

Основная сущность для управления бризером.

**Режимы работы (HVAC Modes):**

- **Off** — выключает питание и закрывает заслонку
- **Heat** — приток воздуха с нагревом до заданной температуры
- **Fan Only** — вентиляция без нагрева с открытой заслонкой

**Управление скоростью вентилятора:**

7 предустановленных скоростей — от 1 до 7

**Preset Modes (позиции заслонки):**

- **Приток** — заслонка открыта, воздух с улицы
- **Смешанный** — смешанный режим
- **Рециркуляция** — заслонка закрыта, воздух из помещения

**Управление температурой нагрева уличного воздуха:**

- Диапазон: 10-30°C
- Шаг: 0.5°C

### Switch

Переключатель питания бризера с автоматическим управлением заслонкой.

- **При включении:** открывает заслонку (приток уличного воздуха) и включает вентилятор
- **При выключении:** закрывает заслонку (рециркуляция) и выключает вентилятор

Идеально подходит для простого управления бризером в сценариях "вкл/выкл" с притоком свежего воздуха.

### Fan

Сущность для управления вентилятором без управления заслонкой.

- Включение/выключение вентилятора
- Управление скоростью в процентах (0-100%) — преобразуется в скорости от 1 до 7
- **Важно:** НЕ изменяет положение заслонки при включении/выключении

Используйте Fan, если хотите управлять только вентилятором, сохраняя текущее положение заслонки.

### Select (Заслонка)

Сущность для независимого управления позицией заслонки.

- **Приток** — заслонка открыта
- **Смешанный** — смешанный режим
- **Рециркуляция** — заслонка закрыта

## Особенности работы

### Switch vs Fan — главное различие

- **Switch** — управляет питанием И заслонкой. При включении всегда обеспечивает приток уличного воздуха.
- **Fan** — управляет ТОЛЬКО вентилятором. Положение заслонки остаётся неизменным.

### Поведение при включении/выключении

- **Switch:** при включении открывает заслонку, при выключении закрывает
- **Fan:** НЕ трогает заслонку ни при включении, ни при выключении
- **Climate:** HEAT/FAN_ONLY открывает заслонку, при выключении закрывает

### Пассивное проветривание

Можно открыть заслонку при выключенном вентиляторе. Это позволяет организовать естественную вентиляцию без шума работающего вентилятора.

**Способ 1 (через Climate):**
1. Выключите climate entity
2. Включите пресет "Приток"

**Способ 2 (через Fan + Select):**
1. Выключите вентилятор через Fan entity
2. Установите заслонку в позицию "Приток" через Select entity

**Способ 3 (через Switch + Select):**
1. Выключите Switch (закроет заслонку)
2. Установите заслонку в "Приток" через Select (откроет заслонку без включения вентилятора)
